> 这是一个随性的Blog.我也不知道要表达什么,也不知道能坚持多久,虽然有时候想写,但是又没有多少时间去写
>
> 写技术文章,感觉,自己水平有限
>
> 写随笔,也没那个文笔
>
> 所以,写意识流,随便写点什么就是了
